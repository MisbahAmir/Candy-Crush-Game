# 🎮 Candy Crush Game (Basic Version)

> 🍬 A simple Candy Crush-inspired web game built to demonstrate basic game logic and interactive UI using JavaScript.

---

## 📌 Description
This project is a basic Candy Crush-style game developed using HTML, CSS, and JavaScript. It features a single playable level where users can match elements and interact with the game board, showcasing fundamental game development concepts.

---

## 🚀 Features
- 🍭 Match similar items (basic mechanics)  
- 🎯 Single level gameplay  
- 🧠 Basic game logic implementation  
- 🎨 Simple and interactive UI  
- ⚡ Lightweight and fast  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---


---

## ⭐ Project Purpose
This project was created to practice and demonstrate fundamental JavaScript concepts such as DOM manipulation, event handling, and basic game logic.
